[判断字符串真实长度]()
------------------
方法如下:
```java
private static int String_length(String value) {
  int valueLength = 0;
  String chinese = "[\u4e00-\u9fa5]";
  for (int i = 0; i < value.length(); i++) {
   String temp = value.substring(i, i + 1);
   if (temp.matches(chinese)) {
    valueLength += 2;
   } else {
    valueLength += 1;
   }
  }
  return valueLength;
 }
```
需要使用,调用该方法即可
